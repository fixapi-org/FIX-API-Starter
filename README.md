# FIX-API-Starter
This is a FIX API based application starter open source repository. We code a typical sample compatible with almost every financial institution.

# Prerequisites
1. Spring boot IDE(Eclipse)
2. FIX API demo account

# Spring Boot Version
2.6.4

# QuickFIXj Version
2.3.0
